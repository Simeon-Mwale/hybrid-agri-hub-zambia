## 🌾 Hybrid AgriHub Zambia
Empowering Zambian farmers with real-time crop prices, USSD/SMS access, and ML forecasting.

### Features
- 📡 Africa's Talking USSD/SMS integration
- 💰 Price alerts for Maize, Soya, Groundnuts, Rice, Cassava
- 🤖 Python/ML price prediction models
- 👨‍🌾 Farmer dashboard (Next.js + TypeScript)

### Local Setup
1. `npm install`
2. Copy `.env.example` → `.env.local`
3. `npx prisma migrate dev`
4. `npm run dev`
